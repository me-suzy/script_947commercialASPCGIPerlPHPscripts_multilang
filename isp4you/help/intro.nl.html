isp4you

isp4you is een hulpmiddel voor het aanmaken van domeinen, mail gebruikers, mysql database, naam server en webmin gebruiker in een stap

Aan de rechterkant kan je kiezen wat je wilt aanmaken. Standaard wordt er een Apache server en een gebruikers account aangemaakt. Nadat je de domein naam ingegeven hebt, zal isp4you automatisch een wachtwoord (8 karakters) aanmaken. Verdere documentatie -->Verdere documentatie
