  1.1 Über phpAdsNew  

  1.1.1 Zusammenfassung  

 


phpAds ist ein Banner-Management- und Banner-Tracking-System, in PHP programmiert.

Funktionen:
- Verwaltung mehrerer Kampagnen pro Kunde
- Verwaltung mehrerer Banner (unterschiedlicher Größen) pro Kampagne
- Tägliche, wöchentliche, vollständige oder zusammengefasste Statistiken
- E-Mailversand der Statistiken an die Kunden
- Unterstützung von HTML- (Rich media), GIF-, JPEG- und PNG-Bannern
- Unterstützung von Bannern auf externen Servern oder
  Speicherung von Bannern über das Administrations-Interface in der
  MySQL-Datenbank oder auf einem FTP-Server.
- Unterstützung von Banner-Netzwerken kombiniert mit eigenen Bannern
- Anzeige von zufälligen, vordefinierten oder HTML-/Grafik-Bannern
- Unterstützung von multiplen Schlüsselworten für Bannergruppen
- Unterstützung von bedingten Schlüsselworten
- Aktivierung und Deaktivierung einzelner Banner
- Bannersteuerung nach: Benutzer, Wochentag, IP-Maske, Domain, Herkunftsseite, Zeit
- Unterstützung von Gewichtung nach Banner oder Kunde
- Targeting von Banner-Klicks auf Zielframes
- Verschiedene Arten von zufälliger oder sequentieller Banneranzeige

 

Sie finden die aktuelle Version unter folgender Internetadresse http://www.sourceforge.net/projects/phpAdsNew/.

 

 
  
  1.1.2 Systemvoraussetzungen  

 


PHP 3.0.8 oder höher (100% PHP4-kompatibel)
MySQL

phpAdsNew sollte unter PHP ab der Version 3.0.8 lauffähig sein, konnte aber bis jetzt aber noch nicht getestet werden. Die getesteten Systeme beinhalten die Versionen 3.0.12, 3.0.18 und 4.0.x. Vorzugsweise sollten PHP 4.0.x-Versionen eingesetzt werden. Die PHP 3 Unterstützung kann in zukünftigen Versionen von phpAdsNew entfallen.

Sollten Sie phpAdsNew auf einem PostgreSQL-Server einsetzen wollen sollten Sie einen Blick auf phpPgAds von Matteo Beccati werfen, welches ein Port von phpAdsNew zu PostgreSQL ist. phpPgAds benötigt PHP4.
Das phpPgAds-Projekt kann unter http://www.greatbridge.org/project/phppgads gefunden werden.
