isp4you

isp4you ist ein Tool um in einem Schritt einen virtuellen Domainaccount, Mailuser, mysql Datenbank, Nameserver, Quoten, Webalizer Support und Webminuser anzulegen. Einfach die Features anklicken die zur Domain angelegt werden sollen. Nachdem der Domainname eingeben wurde und auf den [erstellen] Button geklickt wurde, generiert das Skript automatisch ein zufällig erzeugtes 6,8 oder 10 Zeichen langes Password. Die Daten werden in einer MySQL Datenbank gespeichert.
